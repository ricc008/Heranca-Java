# Estrutura de Dados – Herança em Java

Este projeto tem como objetivo demonstrar a aplicação do conceito de **Herança** em **Java**, utilizando os princípios da **Programação Orientada a Objetos**.

O projeto foi desenvolvido como parte dos estudos acadêmicos na disciplina de **Estrutura de Dados**.

---

## 📚 Conceitos abordados

- Programação Orientada a Objetos
- Herança
- Classes e subclasses
- Métodos
- Encapsulamento
- Java básico
